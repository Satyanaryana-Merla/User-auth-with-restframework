Creating new venv 		: 	python -m venv myworld

To activate venv 		: 	myworld/Scripts/activate.bat

installing Django		: 	pip install django

Creating the folder		:	django-admin startproject projectname

moveing into that folder: 	cd foldername

Creating App 			: 	python manage.py startapp appname

Run server			    :	python manage.py runserver

Any modification		:   python manage.py makemigrations , python manage.py migrate

Database connection		: 	pip install mysqlclient
